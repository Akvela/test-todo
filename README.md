# Todo List

Реализация списка задач. Можно добавить задачу, отредактировать ее и удалить.
Стейт сохраняется при обновлении страницы. В проекте используются технологии:


* Vue 3
* Composition API
* Vuex
* Vue-router
---


#### Команды
```bash
# установка пакета
npm i

# запустить на localhost:8080
npm run dev

# сборка
npm run build
```
